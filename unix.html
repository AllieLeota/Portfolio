Unix Command


In the following presentation, I demonstrate my command of UNIX command prompts, as well as information as to the usefulness of the command prompts. I included two bonus commands at the end of the assigned prompts.

This assignment was very instructive, if for no other reason, than to demonstrate the way we conceptualize information. When we visually browse through a file structure on a desktop computer it's easy to conceptualize how the files and directories are placed within one another, when in reality all the information are simply strings of characters mixed together. With the Unix commands I have utilized thus far I see how the Unix shell attempts to help us conceptualize in that very way, making files and folders browsable. It's interesting to see how the organization system is conceptualized, and how that concept translates to the user through interaction with the server.

Additionally, the process of exploring the Unix server showed me how versatile technical knowledge can be. Even with this elementary knowledge of a Unix server, I can navigate and change items on the web. How cool is that? I can change things that actually show up for other people, on their browser. I can change the experience of a website that others can see and interact with. It's a powerful tool, and I look forward to learning more commands and becoming more proficient with using Unix command lines.
