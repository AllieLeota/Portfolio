Database and Instruction Assignment


My group and I used database optimization tools to recommend changes to a client website. We created an access database for housing new client information. We created a mock-up of an informational page that potential clients would use to enter information into the database. Then we created a Customer Database which contains tools for maintaining and submitting SQL queries to the database, and for creating reports. The following video explains many of the features of the database, the basics of how to use it, as well as information about crafting SQL queries and reports.

In order to effectively implement a renewed social media marketting strategy for our client, we created an instructional presentation on how to create "memes" for use on social media, to capture interest and involvement from technologically savvy potential clients.
