Database and Instructional Design


Team SAKA Collaborators:
Amy Shropshire
Jerome Azbell
Burns Kennedy
Joanna Averch

My group and I used database optimization tools to recommend changes to a client website. We created an access database for housing new client information. We created a mock-up of an informational page that potential clients would use to enter information into the database. Then we created a Customer Database which contains tools for maintaining and submitting SQL queries to the database, and for creating reports. My main contribution was the database design, and the following presentation video explaining many of the features of the database, the basics of how to use it, as well as information about crafting SQL queries and reports.

In order to effectively implement a renewed social media marketting strategy for our client, we created an instructional presentation on how to create "memes" for use on social media, to capture interest and involvement from technologically savvy potential clients. Pandora Database and ER Modeling


Collaborators:
Corinne Chatnik
Daniel Mount
Amy Shropshire
Sunny Zhang

For this project we recreated a simplified version our chosen client database: Pandora Music Service. The Pandora database is obviously too complex to recreate in full, but we created a simplified version to mimic it's functionality. The purpose of this was to demonstrate more in depth knowledge of database design, as well as to find ways to optimize our mock client database for social media and marketting purposes.
